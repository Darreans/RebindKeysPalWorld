PALWORLD ACCESSIBLE CONTROLS v1.0.55 - STEAM

INSTALL
1. Close Palworld.
2. Extract the entire release ZIP.
3. Run "Install Palworld Mod.bat".
4. Start Palworld through Steam.
5. Accept Palworld's normal Mods Detected warning.
6. Open Options.

IN PALWORLD
Keyboard settings:
- Change Weapon Up
- Change Weapon Down
- Rotate Building Left
- Rotate Building Right

Control settings:
- Toggle Aim (ADS)

DEFAULTS AND SAVING
- Change Weapon Up defaults to Mouse Wheel Up.
- Change Weapon Down defaults to Mouse Wheel Down.
- Building rotation remains a separate contextual action and can share the wheel.
- Custom weapon keys save across restarts and update the weapon HUD prompt.
- Clearing one weapon row removes its saved override and restores that wheel direction.

RESTORE MOD DEFAULTS
On the Keyboard page, press F12 Restore Mod Defaults and confirm Yes.
This restores all four mod rows: weapon switching returns to Mouse Wheel Up/Down,
and both building-rotation rows return to Palworld's original defaults.
Other Palworld controls are not changed by F12.

There is no desktop app, tray icon, or overlay to run.

UNINSTALL
1. Close Palworld.
2. Run "Uninstall Palworld Mod.bat".

The uninstaller removes this mod and preserves the shared UE4SS runtime.
